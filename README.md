# OpenStudyGuide

A study-assistant web app that lets you upload PDFs or images, perform OCR text extraction, interact with your material via chat, and export your sessions.

Live demo: https://openstudyguide.onrender.com/

# Features

-   Upload PDF or image files (e.g., handwritten notes, scanned slides) and extract text using OCR.

-   Chat with your uploaded content using AI: ask questions, get explanations, follow-up interactions.

-   Export your conversation or derived study notes for review or offline storage.

-   Simple, intuitive web interface — no heavy setup needed for everyday use.

# Tech Stack

**Front-end:** HTML, CSS, JavaScript

**Back-end:** Python (Flask)

**APIs:** [OpenRouter](https://openrouter.ai/) API, [OCR Space](https://ocr.space/ocrapi) API

**Hosted on:** [Render](https://render.com/)

# Getting Started

```
git clone https://github.com/ayushkumar-005/OpenStudyGuide.git

cd OpenStudyGuide

python3 -m venv venv

source venv/bin/activate   # Windows: venv\Scripts\activate

pip install -r requirements.txt

python app.py
```

Then open http://localhost:5000 (or the appropriate port) in your browser.
